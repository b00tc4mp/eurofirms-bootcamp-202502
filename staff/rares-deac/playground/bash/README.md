## Commands

### pwd

Muestra la ruta del directorio de trabajo

### mkdir

crea un directorio

### ls

enlista archivos y carpetas

### cd

cambia a otra carpeta

### clear (CTRL + L)

borra todo lo escrito en la terminal

### touch

crea un archivo
