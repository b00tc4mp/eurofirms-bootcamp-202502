Commands

* pwd

  Shows the path to working directory.

* mkdir

  Creates a directory.

* cd folder-path

  Changes to another directory (folder).

* touch

  Creates a file.

* CTRL + L

  Clear the screen.

* CTRL + C

  Kill process
